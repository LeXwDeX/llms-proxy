# v1.0.0 发布说明

## 概要
- 本版本为项目首次稳定发布，基于 `dev` 分支的完整功能集构建了可运行的 LLMs Proxy。
- 提供统一的配置、日志与部署脚手架，方便快速落地环境。

## 关键更新
1. **核心代理能力**：新增主程序与认证、日志、中间件等内部模块，串联代理转发全流程。
2. **配置体系**：提供 `.env.example` 与多套 JSON 模板，移除 public base url 约束，支持按环境自定义。
3. **部署工具链**：补充 Dockerfile、docker-compose、systemd 单元与 Makefile，覆盖本地与生产部署场景。
4. **测试与文档**：新增跨平台集成测试脚本、操作手册、内部培训及 API 文档，保证团队协作与质量。

## 升级与部署建议
- 根据目标环境复制并调整 `config/config.production.template.json` 与 `.env.example`。
- 运行 `make build` 或使用 `docker-compose up -d` 快速验证部署。
- 推送标签 `v1.0.0` 至远端后，可在仓库 Release 页面发布正式说明。
